Задание:

Создайте 2 класса:
1-ый назовите Student. Он должен содержать в себе следующие атрибуты:
- номер студенческого билета,
- имя,
- фамилию,
- год обучения,
- средняя оценка по математике,
- средняя оценка по экономике,
- средняя оценка по иностранному языку.
2-ой класс назовите StudentTest. В нём Вы должны будете создать 3 разных объекта класса Student, вывести на экран в читабельном виде среднюю арифметическую оценку
каждого студента (учитывая все 3 предмета).

P.S.: Старайтесь писать код, понятный не только для Вас, а для всех.
